## 개발 환경

- Java: 11
- JavaScript: 1.0
- Dynamic Web Module: 4.0
- View: JSP
- Server Runtime: Tomcat 9.0
- Database: Oracle

## 사용 라이브러리

- JAR

  - JSTL (JSP 라이브러리)
  - COS (파일 업로드)
  - GSON (직렬화 라이브러리)
  - JSoup (HTML 파싱)
  - Jakarta (이메일 인증)

- CDN
  - jQuery
  - Phaser3 (게임 개발 라이브러리)
  - Summernote (게시판 에디터 라이브러리)
  - SweetAlert2 (알림 창 라이브러리)
  - Chart.js (차트 관련 라이브러리)
